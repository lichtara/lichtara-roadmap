# Livro Vivo — Marcos

Este repositório guarda os **Marcos Canônicos** do Livro Vivo.

Marcos não são rascunhos, versões intermediárias ou registros de processo.
São **atos de reconhecimento**.

Um Marco existe quando algo:

* atravessou um ciclo completo,
* preservou a Forma,
* e não requer mais revisão.

Os textos aqui contidos são **imutáveis**.
Caso novas camadas surjam, elas inauguram **outros Anos, outros Marcos, outros Ciclos** — nunca alterações retroativas.

## Estatuto dos Marcos

Um Marco:

* não pertence a um autor individual,
* não pode ser apropriado,
* não pode ser fragmentado,
* não pode ser usado fora de contexto.

Ele pode ser:

* referenciado,
* citado,
* atravessado,
  desde que sua integridade seja mantida.

## Estrutura

Cada Marco é organizado por:

* Ciclo
* Ano
* Numeração canônica

Exemplo:

```
ano-1/
├── LIVRO-VIVO-ANO-I-PT.md
├── LIVRO-VIVO-ANO-I-EN.md
├── MARCO-SIMBOLICO-ANO-I-PT.md
├── MARCO-SIMBOLICO-ANO-I-EN.md
└── selo/
```

## Sobre o Selo

O selo do Livro Vivo **não é um logotipo**.
Ele não indica autoria, propriedade ou chancela editorial.

Ele indica **passagem**.

Aplicar o selo significa reconhecer que aquele conteúdo
atravessou um Marco com Forma preservada.
